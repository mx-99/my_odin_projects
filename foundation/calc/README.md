# Calculator

A simple web-based calculator built using HTML, CSS, and JavaScript. It supports basic arithmetic operations including addition, subtraction, multiplication, and division. Additionally, it features a sum function that calculates the sum of all digits input by the user.

## Features

- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Sum function to calculate the total of all input digits
- Keyboard support for entering numbers and operations
- Clear button to reset the calculator

## Usage

1. **Open `index.html`** in your web browser to use the calculator.
2. **Click on the buttons** or use your keyboard to enter numbers and operations.
3. **Press `=`** to calculate the result of the arithmetic operation.
4. **Press `C`** to clear the input and start a new calculation.
5. **Use the `Σ` button** to compute the sum of all input digits.

## Keyboard Shortcuts

- **Numbers (0-9)**: Enter numbers into the calculator.
- **Operators (`+`, `-`, `*`, `/`)**: Perform the corresponding arithmetic operations.
- **`Enter`**: Calculate the result of the current operation.
- **`Backspace`**: Remove the last digit or character from the input.
