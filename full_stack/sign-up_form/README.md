# Sign-up Form

Simple sign-up form built with HTML, CSS, and JavaScript. 

## Features

- Ok layout
- Custom font integration
- Basic form validation for password confirmation

## Technologies Used

- **HTML**: Structure of the form
- **CSS**: Styling the layout and appearance
- **JavaScript**: Client-side form validation
